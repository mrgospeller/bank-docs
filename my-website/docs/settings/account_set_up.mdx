# Настройка аккаунта

Эта инструкция поможет вам настроить ваш аккаунт в приложении **Nova Bank**.

1. Введите ваш адрес электронной почты

<img src="/my-website/docs/settings/account_set_up_1.png" alt="Экран ввода электронной почты" width="321" height="696">

2. Выберите тип учётной записи

<img src="/my-website/docs/settings/account_set_up_2.png" alt="Экран изменения типа учётной записи" width="321" height="696">

3. Добавьте адрес регистрации и почтовый индекс 

> Проверьте, что данные совпадают с вашим документом!

<img src="/my-website/docs/settings/account_set_up_3.png" alt="Экран добавления адреса" width="321" height="696">

4. Добавьте личные данные — полное имя и дату рождения

<img src="/my-website/docs/settings/account_set_up_4.png" alt="Экран добавления личных данных" width="321" height="696">

Вы можете поменять настройки в любой момент в приложении **Nova Bank**.
